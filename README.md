# Color Confident Studio

**Color Confident Studio** is a mobile-first web app that bridges digital color inspiration and real embroidery materials. Crafters can browse actual thread colors, build a digital stash that mirrors their physical collection, create beautiful palettes, preview combinations in sample projects, and plan their next purchase.

## Technical stack

- **Framework:** Ruby on Rails 8
- **Database:** PostgreSQL
- **Frontend:** Hotwire (Turbo + Stimulus)
- **Styling:** Tailwind CSS with daisyUI
- **Authentication:** Rails built-in authentication

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/karenbarbe/color-confident-studio.git
   ```

2. Navigate to the project directory:
   ```
   cd color-confident-studio
   ```

3. Install the required gems:
   ```
   bundle install
   ```

4. Set up the database:
   ```
   rails db:setup
   ```

5. Start the Rails server:
   ```
   bin/dev
   ```

## Usage

1. Start the Rails server:
   ```
   bin/dev
   ```

2. Open your browser and navigate to http://localhost:3000

3. Create an account to start building your color stash
